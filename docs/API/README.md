# 全局对象

## 默认导出 Complex

Complex 是 `math-complex` 的默认导出模块，你可以这样使用它

``` javascript
import Complex from "math-complex";

const { 
    realize, 
    imagine, 
    add, 
    mult, 
    abs, 
    contrary 
} = Complex.Reflect;
```

Complex本身就已经包含了`math-complex`的所有导出对象，不过一些是采用了别名。
